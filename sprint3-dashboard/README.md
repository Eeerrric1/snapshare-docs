# Dashboard da Sprint 3 — SnapShare

Este pacote contém um dashboard interativo implementado em **Streamlit**.


## Arquivos
- `app.py` — aplicação Streamlit (rodar com `streamlit run app.py`).
- `data/sample_data.csv` — dados usados no dashboard.
- `requirements.txt` — dependências, instalar com `pip install -r requirements.txt`.


## Como executar
1. Recomendo criar e ativar um ambiente virtual: 
- Para instalar: `python -m venv .venv`
- Para ativar: `.\.venv\\Scripts\\activate` no Windows, `source .venv/bin/activate` no Linux/Mac.

2. Instale dependências:
- `pip install -r requirements.txt`

3. Rode o dashboard:
- `streamlit run app.py`

4. O Streamlit abrirá uma aba no navegador com o dashboard.
